通常导入模块的路径都在sys.path中，可以将自己的需要的目录用append添加到其中

第一次导入模块时，模块会执行其中的语句，第二次时则不会
因为模块通常用于定义函数，变量，类等，在第一次导入时就已经执行定义过了，所以不需要第二次执行
可以通过importlib模块的reload重新载入模块，只接受一个模块为参数

在模块中添加测试代码
__name__属性只有在主程序中会显示__main__
因此，可以使用if __name == '__main__'来检测是否在主程序中，如果时，就执行测试代码

模块放哪里能被使用：
1.sys.path包含的路径中
2.添加到pythonpath中？

包：
特点：
1.包是含有__init__.py文件的目录
2.包可以包含很多模块
3.在__init__.py中的内容，可以用包名.的方式直接使用

dir:查看对象的所有属性，返回一个列表

__all__:在模块中定义的，最好使用其中包含的内容
在from 模块 import * 中，只能得到__all__中的内容，这也是编写模块的作者希望的，
如果要用其他的内容，必须显示的指明
