windows下
在官网下载所需版本mysql
地址：https://dev.mysql.com/downloads/mysql/
解压后放在c盘根目录
使用mysqld --initalize-insecure --user=mysql --consol;
创建my.ini文件，对其进行配置
例子：

[mysqld]
#设置mysql的安装目录
basedir=C:/mysql-8.0.13-winx64
#设置mysql数据库的数据的存放目录
datadir=C:/mysql-8.0.13-winx64/data
#设置默认使用端口
port=3306
#允许最大连接数
max_connections=200
#允许连接失败的次数。这是为了防止有人试图攻击数据库
max_connect_errors=10
#服务端使用的字符集
character-set-server=utf8mb4
#数据库字符集对应一些排序等规则使用的字符集
collation-server=utf8mb4_general_ci
#创建新表时使用的默认存储引擎
default-storage-engine=INNODB
#默认使用mysql_native_password插件作为认证加密方式
#mysql8.0默认认证加密方式为caching_sha2_password
default_authentication_plugin=mysql_native_password

[mysql]
#设置mysql客户端默认字符集
default-character-set=utf8mb4

[client]
default-character-set=utf8mb4
port=3306

添加到系统环境
将bin目录地址加入环境变量

添加时区表
windows下不自带时区表，需要在官网下载
地址：https://dev.mysql.com/downloads/timezones.html
添加时区表
mysql -u root -p mysql < timezone_posix.sql
