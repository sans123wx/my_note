html5学习笔记

概念

html标签：布局网页的结构
空标签：只有标枪，没有内容
自闭和标签：空标签可以这样表示
虚标签：两种表示方法：

属性

标签属性：只能用在开始标签或单个标签上
标签属性分为全局属性，私有属性，属性与属性之间用空格隔开
以data-开头的叫自定义属性或者扩展属性

元素关系

<!doctype html>
<html>
<head>
<title>标题</title>
</head>
<body>
<code>内容</code>
</body>
</html>
html是head的父元素
head是html的子元素
html与body是兄弟元素
title是html的后代元素

css属性

css属性：设置内容的样式
几种使用css的方法
1. 用属性：style直接在元素上定义，style = ‘background-color:grey;color:white’
2. 使用标签<style>：一般在head中使用
<style type = ‘text/css’>
a{background-color:grey;
color:white}
</style>
3. 使用外部样式表，用link元素导入
<link rel = ‘stylesheet’ type = ‘text/css’ href = ‘style.css’>
后面导入的样式会覆盖前面的，如果使用！important，则优先使用！importtant标记的属性
color:black !important
！important>属性style>元素style>link元素
